# School Management System (SMS)

This is a comprehensive school management system built with React, Vite, and Firebase.

## Features
- Admin Dashboard
- Staff Management
- Student Management
- Class Management
- Course Management
- Real-time Updates via Firestore

## Getting Started
To run the project locally:
1. `npm install`
2. `npm run dev`

## Technologies
- React
- Vite
- Firebase (Firestore, Auth)
- CSS (Modern, Responsive)
